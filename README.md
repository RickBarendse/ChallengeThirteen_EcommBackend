# eCommerce Backend
 
## Description
The backend for an eCommerce website that uses Express.js API and Sequelize to interact with a mysql databsase.

## Installation
Application requires the installation of Express.js, Sequelize, and mysql.
  
## Usage
Once the required modules have been installed, use mysql to create the database.  Then use the command line in the integrated teriminal to seed the database using the provided seed files.  After seeding ans staring the application using 'npm start', the user can use Insomnia to interact with the various API routes.
